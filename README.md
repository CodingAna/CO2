# CO2

## Installation

### Set up vuetify to the project
Change to the project directory and run the vue-command.
```
vue add vuetify
```

### Run the webserver
```
npm run serve
```
Note: This serving mode takes longer to run due to loading every module.

### Run the webserver for production
```
npm run build
```
